# Hi, I'm Justin 👋

I am a junior developer in training, currently building practical projects with **Python**, **JavaScript**, and **TypeScript**. My work focuses on web applications, frontend interfaces, data organization, search/filtering experiences, and portfolio-focused side projects.

My background is related to **Library and Information Science**, so I am especially interested in how information can be described, classified, searched, preserved, and presented. I try to bring concepts such as metadata, information organization, digital archives, bibliographic data, and knowledge management into the software I build.
